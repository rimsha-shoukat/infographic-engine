# Infograph 📊
A Data Storytelling Engine — animated charts built from scratch, no libraries.

## What is Infograph?

Infograph turns raw numbers into animated visual stories. You pick a dataset, and watch it come alive — smooth, colorful, interactive charts built entirely with SVG. No Chart.js. No D3. Just code.

---
## Features

- 📈 **3 Chart Types** — Bar, Line, and Pie charts
- 🎬 **Animated** — Every chart animates on load, built with pure SVG
- 📚 **9 Data Stories** — Real-world datasets like rainfall, screen time, market share, and more
- 🔄 **Switch Views** — Toggle between chart types on any story
- 🌙 **Dark Theme** — Clean, minimal dark UI
- 📱 **Responsive** — Works across screen sizes

---

## Tech Stack

| Technology | Purpose |
|------------|---------|
| **Node.js + Express** | Backend server |
| **EJS** | HTML templating |
| **SVG** | Chart rendering & animations |
| **TAILWIND** | Styling & layout |
| **Vercel** | Deployment |

## What I Learned

This project was built specifically to understand SVG from the ground up.

- How SVG **paths, viewBox, and coordinate systems** work
- How to **animate SVG elements** with CSS and JS — no libraries
- Using **EJS templating** with Express for the first time
- Structuring a **Node.js project** cleanly for deployment
- Deploying to **Vercel** from a Node/Express app

## Getting Started

Want to run it locally? Here's how:
```
# 1. Clone the repo
git clone https://github.com/rimsha-shoukat/infographic-engine.git

# 2. Go into the folder
cd infographic-engine

# 3. Install dependencies
npm install

# 4. Build Tailwind CSS
npm run watch:css

# 5. Start the server
npm run dev
```
Then open your browser and go to → `http://localhost:3000`
